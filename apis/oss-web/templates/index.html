OSS web direct upload---Signing in php on the server side, OSS will call back the callback url set by the user when the file is uploaded successfully

  1. Based on plupload package
  2. Support html5,flash,silverlight,html4 and other protocols for uploading
  3. Can run on PC browser, mobile browser, WeChat
  4. Signature is done on the server (php), safe and reliable, recommended!
  5. Show upload progress bar
  6. You can control the size of the uploaded file and allow the type of the uploaded file. In this example, only jpg, png, gif endings and zip, rar files can be uploaded, and the maximum size is 10M
  7. The most important thing is that you can transplant it to your system within 10 minutes and realize the above awesome functions!
  8. Note: the bucket must be set with Cors (Post tick), otherwise there is no way to upload
  9. Note: This example is uploaded to the user-dir directory by default. The setting of this directory is in php/get.php, $dir variable!
  10. Note: Change the callbackUrl variable in php/get.php to your own url
  11. Note: the success returned here is that the OSS has called back the application server, and the application service has returned 200!
